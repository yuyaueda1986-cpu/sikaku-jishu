
# 資格試験クイズアプリ

隙間時間にスマホで解ける、資格勉強用の4択・5択クイズサイト。

## 技術構成

- GitHub Pages による静的サイトホスティング
- HTML / JavaScript / CSS のみで構成（サーバーサイド不要）

## 機能

### クイズ

- 4択・5択の選択式クイズ
- **一問一答モード**: 1問ずつ回答・解説を確認
- **全問回答モード**: 全問回答後にまとめて結果を確認
- 出題順のランダム化（オプション）
- 選択肢のランダム化（オプション）

### 解説

- 各問題に簡単な解説を表示
  - 例: 「X²+Y²=Z² は三平方の定理により成立します。」
- AI解説への誘導テキストを用意し、コピー可能にする
  - 例: 「X²+Y²=Z² は三平方の定理により成立するとあるが、詳しく解説して」
  - ユーザーがこのテキストをコピーし、各自のAIツールで詳細を調べる想定

> **注**: API コスト削減のため、アプリ内でのAI解説機能は搭載しない。

## 制約事項

- ユーザー認証なし
- 回答記録の保存なし
- クイズ内容は固定（ユーザーによるカスタマイズ不可）
- WEBページリロード時は初期状態に戻る（ユーザにリロードしないよう、注意喚起しておく）
